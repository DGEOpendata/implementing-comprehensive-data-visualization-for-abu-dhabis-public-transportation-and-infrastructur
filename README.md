# Data Visualization for Abu Dhabi Public Transportation

## Overview
This project provides a comprehensive visualization of Abu Dhabi's public transportation and infrastructure data. The visualizations are designed to aid urban planners, transportation agencies, and the general public in understanding transit patterns and infrastructure development.

## Prerequisites
- Python 3.x
- Pandas
- Matplotlib
- Seaborn
- CSV and Excel files for the data

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-repository-url
   
2. Install the required Python packages:
   bash
   pip install pandas matplotlib seaborn
   

## Usage
1. Load the data files `public_transportation_routes.csv` and `infrastructure_projects.xlsx` into the `data` folder.
2. Run the Python script:
   bash
   python visualize_transportation.py
   
3. Observe the generated visualizations that will appear, showcasing insights into bus routes and infrastructure projects.
